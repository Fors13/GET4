Программа для заполнения данных в Меркурии из файла.
Файл должен обязательно содержать столбцы: 
  Название шаблона
  Номер машины	
  Номер прицепа	
  Особые отметки	
  Cерия защищенного бланка	
  Номер защищенного бланка
  Номер записи склад.журнала
  Партия
  Документ-образец

После запуска программа откроет новый браузер и попросит авторизоваться.
После авторизации начнет выполнять алгоритм полностью повторяющий человеческие действия
